What changed across every option

  • Agent avatar shrunk: 14px sparkle → 11–12px status glyph. In most options the vendor sparkle is removed entirely; the state glyph carries the signal.
  • No card-in-card. The inner "AGENTS" box with its own border is replaced by a rail, divider, inset panel, tree, or bare list.
  • Indent aligned under the task title, not offset between icons.
  • Row height: 22–24px instead of 30+.

My pick

A (rail) or B (flush + dot-only). A still communicates "these belong to the parent" visually. B is the leanest — best if users rarely need to distinguish agent vendor at the child level.
